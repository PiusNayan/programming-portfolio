# 📖 Key Concepts Learned

## What is a Program?

A program is a sequence of instructions that tells a computer how to perform a task.

---

## What is Python?

Python is a high-level, general-purpose programming language known for its readability, simplicity, and wide range of applications such as:

- Web Development
- Automation
- Data Science
- Artificial Intelligence
- Cybersecurity
- Machine Learning

---

## Variables

Variables are named locations in memory used to store data.

Example:

```python
name = "Yaw"
age = 21
```

---

## Data Types

| Data Type | Description | Example |
|-----------|-------------|---------|
| int | Whole numbers | 25 |
| float | Decimal numbers | 3.14 |
| str | Text | "Hello" |
| bool | True or False | True |

---

## Input and Output

### Input

```python
name = input("Enter your name: ")
```

### Output

```python
print(name)
```

---

## Type Conversion

Converting user input into another data type.

Example:

```python
age = int(input("Enter your age: "))
```
